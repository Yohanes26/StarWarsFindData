THE UX JEDI

Essa Aplicação Web tem a finalidade de disponibilizar para qualquer público, as informações de todos os personagens da Saga Star Wars e quantos personagens possui em cada Filme. Para isso, utilizamos a API chamada SWAPI (https://swapi.co/documentation), levando em consideração nossos

conhecimentos UX Jedi, procuramos fazer um layout que seja responsivo.

Para te ajudar a navegar, temos uma NavBar acima, onde temos 3 itens de navegação:


1) É o próprio "Home", para você poder voltar aqui e rever as orientações.

2) "Personagens", onde você terá acesso a todos os personagens da Saga Star Wars.

3) "Filmes", Nessa página conseguimos ver as estatísticas de quantos personagens possui em cada filme da Saga.
